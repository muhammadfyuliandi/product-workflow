# product-workflow

AI-powered product doc pipeline for VS Code.

**Idea → Discovery → PRD → Roadmap → Tech Spec → Tasks**

## Setup
1. Clone this repo
2. Open in VS Code with GitHub Copilot installed
3. Open Copilot Chat (`Ctrl+Shift+I`)
4. Pick an agent from the dropdown or type `/feature-brief`

## Docs
- `docs/product/` — Discovery, PRD, Roadmap
- `docs/engineering/` — Tech Spec, Task Breakdown

## Agents
| Agent | Trigger |
|-------|---------|
| Product Discovery | Agent dropdown |
| PRD Writer | Agent dropdown or `/generate-prd` |
| Tech Spec Writer | Agent dropdown |
| Task Breakdown | Agent dropdown or `/generate-tasks` |
